Projeto Final VR - Cidade RPG

Descrição:
Este projeto foi desenvolvido na Unity utilizando Meta XR SDK e OpenXR como parte do projeto final introdutório de Realidade Virtual.

A cena apresenta um ambiente RPG explorável contendo personagens, dragão animado, sistema de câmera, interação com dado D20 e elementos urbanos organizados em um cenário virtual.

Funcionalidades:
- Movimentação no PC
- Controle de câmera
- Ambiente 3D interativo
- Física aplicada em objetos
- Sistema de dado D20
- Personagens animados
- Configuração para Meta Quest / Android
- OpenXR e Meta XR SDK configurados

Tecnologias Utilizadas:
- Unity 6
- C#
- Meta XR SDK
- OpenXR

Estrutura do Projeto:
- Assets
- Packages
- ProjectSettings

Como Executar:
1. Abrir o projeto na Unity 6
2. Carregar a cena principal
3. Pressionar Play para testar no PC
4. Para VR, configurar dispositivo Meta Quest via Android Build
